== Description ==

Este plugin crea un post personalizado de tipo portafolio en el que se pueden crear portafolios o proyectos. Se pueden agregar enlaces de GitHub, enlaces de Sitio en Vivo, Imágenes del Portafolio para una galeria e Iconos del portafolio para describir las tecnologías utilizadas con el mismo .

== Installation ==

1. Upload `portfolio-plugin` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[portfolio_posts]` shortcode to display the portfolio.
